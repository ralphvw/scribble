# scribble
Script for cheating at skribble ;)

# Usage
Command Line prompt:

'python3 scribble.py {length of word}' (eg. python3 scribble.py 10)

Inputs:

Enter letter and position as you get clues (eg. a3)

0 to reset

! to quit
